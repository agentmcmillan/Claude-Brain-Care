# Shared Memory Conflicts

Log of merge conflicts from shared memory imports.

## How to Resolve

1. Review each conflict below
2. Choose resolution: `local`, `remote`, or `merge`
3. Mark as resolved with `[x]`
4. Run `/shared-memory sync` to apply

## Conflict Format

When conflicts occur, they appear as:

```markdown
## [timestamp] - [source]

### Remote Context
- Focus: [what remote was working on]
- Key files: [list]

### Local Context
- Focus: [what local was working on]
- Key files: [list]

### Resolution
- [ ] Keep local (discard remote)
- [ ] Accept remote (overwrite local)
- [ ] Merge both (combine contexts)

**Chosen**: [pending]
**Resolved by**: [user]
**Date**: [date]
```
